Key Projects

OasisAI - The 6.6-Gigapixel Desert

This project explores the limits of standard hardware and Generative AI. I built and trained a DCGAN from scratch on a home CPU for 1,000 epochs to generate unique desert landscapes. When attempting to stitch 100,000 of these generated images into a single massive mosaic, I hit a severe RAM bottleneck (MemoryError). To solve this, I utilized memory-mapping in Python to bypass the RAM entirely, writing a 6.6-Gigapixel (20GB) image directly to the hard drive. The final result is a video featuring an infinite exponential zoom into an AI-generated universe.

Infinite Winter – The AI Snowflake Mosaic

This project explores the fascinating intersection of nature's mathematics and Generative AI, built entirely on standard home hardware. The journey began with building and training a Generative Adversarial Network (GAN) from scratch on a basic CPU. The goal? Teaching the AI to understand and generate highly detailed, perfect snowflake fractals. Once the model was trained, it was pushed to its absolute limits to manufacture a massive dataset of unique, AI-generated snowflakes. The primary engineering challenge was stitching these hundreds of thousands of generated images into a single gigapixel-scale mosaic without triggering a fatal memory crash (MemoryError). To bypass the computer's limited RAM, a custom Python script was engineered utilizing dynamic Memory-Mapping, which writes the massive image data directly to the hard drive in strict batches. The final result is a hypnotic video featuring an exponential, infinite "deep zoom" into a vast digital winter, where getting closer reveals that every single pixel is actually another fully unique, AI-generated snowflake.

The AniMosaic Project – Custom AI Training & 4.4 Gigapixel Mosaic

An experimental hardware and software project designed to push the absolute limits of a repurposed, weak PC. The project began by training a custom AI image generation model entirely from scratch. Once trained, the goal was to push the hardware to the extreme by generating 20,000,000 individual anime faces and seamlessly stitching them together into a single, massive 4.4 Gigapixel mosaic. To prevent the hardware from crashing due to memory (MemoryError) and heavy I/O operations, a custom Python script was developed to handle the files using strict batch processing. Intermediate tiles were saved directly to the hard drive, keeping the RAM clear. The machine ran at 100% CPU capacity for 48 hours straight to successfully render the final image.
